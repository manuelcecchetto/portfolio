The
Gravity
Sorter

A machine for organizing shadows by weight. It functions only when the observer refuses to look directly at it. If a shadow escapes, it must be bribed back with static.
Process Notes
Built using discarded physics engines and a collection of 404 errors. We found that gravity is more of a suggestion than a rule when you code in the dark. The primary challenge was preventing the buttons from feeling too much like buttons.
